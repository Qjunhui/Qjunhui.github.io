正则表达式

正则表达式也叫规则表达式，就是用来匹配字符串的。由基本的元字符和限定字符组合成的。

一个表达式，然后这个表达式有自己的规则。

作用：通过这个规则验证字符串，从而获取想要的内容

基本元字符

1). 除了\n以外的任意的单个字符（字符串）

\r \n 回车换行

2)[ ] 表示的范围

[ . ]    就是一个点

[m-n]    表示m到n之间（包括m，n）任意的一个数字

         最大范围是[0-9]

eg：   [a-z]             表示的是所有的小写字母中的任意一个

       [A-Z]             表示的是所有的大写字母中的任意一个

       [a-zA-Z]            表示的是所有的字母中的任意一个

       [0-9a-zA-Z]       表示的是所有的数字和字母中的任意一个

3)（） 提升优先级；分组

分组：一对小括号就是一个组，从左到右计算组的个数

4)| 或者

级别最低

限定符（+ * ？）

5)+

+前面的表达式出现1次到无限次

6)*

*前面的表达式出现0次到无限次

7)?

?前面的表达式出现0次到1次

阻止贪婪模式

8){ } 表示前面表达式的出现次数

{m,n}    表示大括号前面的表达式最少出现m次，最多出现n次

{0, }    表示大括号前面的表达式最少出现0次，最多出现无限次

{ ，5}   错误写法！！！

9)\d 表示的是所有数字中的一个======等价于[0-9]

\D 表达的是所有的非数字中的一个

10)\s 表达的是空白符号

\S 表达的是非空白符号

11)\w 表示的是非特殊符号(不严谨)

数字，字母，下划线

\W 表示的是特殊符号(不严谨)

12）^ 以什么开头

$ 以什么结尾

^[0-9]    这个字符串必须以数字开头

[^0-9]    取非

[a-z]$   这个字符串必须以小写字母结尾

创建正则表达式对象

.test () 只要字符串中有与正则表达式匹配的，结果就为true

/^[0-9a-zA-Z_.-]+[@][0-9a-zA-Z_.-]+([.][a-zA-Z]+){1,2}$/
